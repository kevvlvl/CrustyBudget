# Crusty Budget App

This simple CRUD app is made using the following tech stack:

- React Vite front-end using Mantine components
- Rust web backend using Axum Tokio and redb key-value data store

It offers the following capabilites:

Manage your budget by entering your incomes, expenses, and generate a report to show how it looks for the your desired time frames (past month, 3 months, YTD, 1 year)
